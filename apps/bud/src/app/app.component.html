Apprenticeship Management Software - Training & Delivery

This is a demonstration of creating a production-ready Angular application using Nx, NgRx, or SignalStore. It leverages a modular UI library and implements software architecture principles such as Domain-Driven Design (DDD).

The purpose of this demo is to showcase how to build scalable, modular, and maintainable applications, aligned with the requirements outlined in the test.

The application features a dynamic form with fields for Name, Email, and a dynamic list of Phone Numbers. It supports real-time validation, persists form data to the global state, and demonstrates modern state management techniques suitable for production environments.
